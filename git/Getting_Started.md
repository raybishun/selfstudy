# Git Basics

### Create a Repository
- https://github.com/

### Download Git Client
- https://git-scm.com/

### Get Git version
- git --version

### Clone a Repository (locally)
1. mkdir scm
2. cd scm
3. git clone https://github.com/raybishun/selfstudy.git

### Git Commit
1. git status
2. git add .
3. git commit -m "Describe the commit..."
4. git push
