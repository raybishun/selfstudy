# ASP.NET Core SE

### Basic dotnet commands
- dotnet --version
- dotnet --info

### Create a new ASP.NET Core Web App
1. mkdir aspnetcoresample
2. cd aspnetcoresample
3. dotnet new web
4. dotnet run
5. http://localhost:5000
